CONNECTION LOCAL REPOSITORY WITH GITHUB

open the folder with which you want to connect the git repository and then apply these commands in command-line :
1> git init
2> git commit -m "first commit"
3> git remote add origin https://github.com/Khazaina/Task01.git
4> git branch -M main
5> git push -u origin main


HOW TO COMMIT AGAIN

1> git add .
2> git commit -m "second commit"
3> git push
